১. AdGuard Home-এর Query Log চেক করুন (সবচেয়ে সহজ)
যেহেতু আপনি মাত্রই AdGuard Home সেটআপ করেছেন, তাই এটি আপনার জন্য সবচেয়ে বড় গোয়েন্দা হিসেবে কাজ করবে।

AdGuard Home ড্যাশবোর্ডে গিয়ে Query Log ট্যাবে যান।

ডানপাশের সার্চ বক্সে 192.168.22.179 লিখে ফিল্টার করুন।

দেখুন এই ডিভাইসটি ইন্টারনেটে কী কী ওয়েবসাইটে হিট করছে। যদি লগে কোনো সুনির্দিষ্ট অফিসের পোর্টাল, কাজের ডোমেইন বা নির্দিষ্ট ফেসবুক/ইউটিউব অ্যাকাউন্টের অ্যাক্টিভিটি পপ-আপ করে, তবে ব্যবহারকারী কে হতে পারে তা ধারণা করা সহজ হবে।

অনেক সময় AdGuard Home স্বয়ংক্রিয়ভাবে ডিভাইসের Hostname (যেমন: Monir-Laptop বা iPhone-of-Sultana) ডিটেক্ট করে আইপির পাশে ব্র্যাকেটে দেখিয়ে দেয়।

২. মিক্রোটিক উইনবক্স (Winbox) চেক করুন
মিক্রোটিক রাউটার তার নেটওয়ার্কে কানেক্টেড হওয়া প্রায় প্রতিটি ডিভাইসের হোস্টনেম রেকর্ড করে রাখে।

Winbox ওপেন করে IP -> DHCP Server-এ যান।

Leases ট্যাবে ক্লিক করুন।

তালিকায় 192.168.22.179 আইপিটি খুঁজুন।

ওই লাইনে Active Host Name কলামটি খেয়াল করুন। সেখানে সাধারণত পিসি বা মোবাইলের নাম লেখা থাকে (যেমন: DESKTOP-R5K2N9, Galaxy-S24, বা ব্যবহারকারীর নিজের দেওয়া পিসির নাম)।

৩. টার্মিনাল থেকে Nmap বা Nbtstat কমান্ড ব্যবহার করুন
আপনার ডকার সার্ভার বা নিজস্ব পিসির কমান্ড প্রম্পট/টার্মিনাল থেকে ডিভাইসটির নেটওয়ার্ক নেম জানার চেষ্টা করতে পারেন।

উইন্ডোজ পিসি থেকে (Command Prompt):

DOS
nbtstat -A 192.168.22.179
যদি ডিভাইসটি কোনো উইন্ডোজ পিসি হয় এবং নেটওয়ার্ক শেয়ারিং অন থাকে, তবে এই কমান্ডটি সরাসরি ওই পিসির ইউজারনেম এবং কম্পিউটার নেম স্ক্রিনে প্রিন্ট করে দেবে।

লিনাক্স সার্ভার থেকে (Nmap টুল ব্যবহার করে):

Bash
sudo apt install nmap -y  # ইনস্টল করা না থাকলে
sudo nmap -O 192.168.22.179
এটি আপনাকে বলবে ডিভাইসটি উইন্ডোজ, লিনাক্স নাকি অ্যানড্রয়েড/আইফোন ব্যবহার করছে, যা আপনাকে ডিভাইসটির মালিক অনুমান করতে সাহায্য করবে।

৪. ম্যাক অ্যাড্রেস (MAC Address) দিয়ে ব্র্যান্ড চেক করুন
Winbox-এর Leases ট্যাব থেকে ওই আইপির MAC Address-টি কপি করে নিন (যেমন: 74:E5:43:XX:XX:XX)। এরপর macvendors.com বা এই জাতীয় কোনো ফ্রি সাইটে গিয়ে পেস্ট করলে এটি বলে দেবে ডিভাইসটি কোন ব্র্যান্ডের (যেমন: Apple, Samsung, Intel, নাকি HP)। নেটওয়ার্কে কার কাছে ওই ব্র্যান্ডের ডিভাইস আছে, তা মিলিয়ে আপনি মালিককে সনাক্ত করতে পারবেন।
